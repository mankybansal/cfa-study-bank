# CFA Study Bank

React + TypeScript + Vite study app with 1000 original CFA-style MCQs across Levels I, II, and III.

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS v4 + shadcn/ui components
- Zustand for local state
- localStorage persistence for progress/session resume
- Vitest + React Testing Library + Playwright

## Features

- 1000-question bank (`L1/L2/L3`) with topic metadata and explanations
- High volume of computational MCQs (TVM, returns, statistics, CAPM, duration/convexity, forwards, hedging, information ratio)
- Practice session setup (levels + question count)
- Exam-style single-answer MCQ flow
- Immediate feedback with explanation
- Local progress tracking (completed sessions, topic accuracy, difficulty coverage)
- Resume-friendly state persistence in browser storage

## Content note

Questions are original and generated from curriculum-aligned topic structures and concepts.
No proprietary CFA Institute question text is copied.

Source references used for 2026 alignment are documented here:
- `docs/sources-2026.md`

## Project structure

- `src/data/questionBank.ts`: deterministic 1000-question generator
- `src/lib/quiz-engine.ts`: session, filtering, scoring logic
- `src/lib/storage.ts`: localStorage persistence
- `src/store/use-study-store.ts`: Zustand app state/actions
- `src/App.tsx`: UI and workflow
